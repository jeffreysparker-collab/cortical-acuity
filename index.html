Method: Pointer & Hess (1989) replication. Horizontal meridian only. Landolt C / Tumbling E: true 4AFC, gap/tine direction random and independent of position. Gabor: 2AFC tilt discrimination (±45°), full contrast. Glass patterns: 2AFC CW vs CCW spiral, coherence staircase (V2/V4 integration). Dipole separation scales with eccentricity using personal k₂ from Landolt (or population k₂=1.5° as fallback). 2-up/1-down staircase → ~70.7% threshold. Stim 200ms to limit microsaccades. Threshold = mean of last 8 reversal points. Schwartz / E₂ parameters from robust Huber linear regression of T(e) = (e+k₂)/k₁. Dashed = P&H population reference (k₁≈25, k₂≈1.5°).
